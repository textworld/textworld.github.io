Monit my tiny vps in aws lightsail

vps配置: 单核 512mb, 月付3.5刀,全天速度不错,不像linode晚上会爆炸,aws lightsail网页管控界面非常清爽.
整体采用node_exporter+prometheus+grafana的方案进行监控. 其中node_exporter通过二进制直接运行,prometheus和grafana通过docker方式部署.

node_exporter

按照官方教程做即可

promethues in docker

我不喜欢用latest这个标签,因为我不知道怎么查latest到底是哪个版本

文件 /etc/prometheus/prometheus.yml写入一下内容,和官方默认的相比是增加了’node’这个爬取任务

grafana in docker

在grafana中导入“a Node Exporter 0.16+ 监控展示看板”这个dashboard,数据源选择prometheus即可.
