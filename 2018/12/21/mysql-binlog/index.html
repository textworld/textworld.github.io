执行insert语句产生的event事件

执行语句INSERT INTO trow VALUES(1, NULL), (2, ‘a’)，这个语句会产生一个query event，一个table_map event、一个write_rows event以及一个xid event。

mysql-mysql数据迁移的测试标准

最基本

  1. 源数据表插入数据, 目的数据表也能插入数据
  2. 源数据表修改对应的数据, 目的数据表也能相应修改对应行的数据
  3. 源数据表删除对应的数据, 目的数据表也能删除对应行的数据

三种模式

  1. 全量
  2. 增量
    • insert
    • delete
    • update
  3. 全量+增量

带where条件的sql语句执行

  1. 从源表中取数据时带有where条件
  2. 通过 由于字段映射,从源表中执行的sql带有 select username as name这样的语句,字段的别名就是迁移到目标表的别名
    • insert 通过
    • delete 通过
    • update 通过
